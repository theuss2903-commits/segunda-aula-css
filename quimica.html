O que é um elemento

Um elemento químico é uma substância formada por átomos que possuem o mesmo número de prótons.

Grupos e Períodos

Hidrogênio (H)

O elemento mais abundante do universo.

Oxigênio (O)

Essencial para a respiração.

Carbono (C)

Base de toda a vida orgânica.

Elementos em destaque

A Tabela Periódica organiza os elementos por número atômico, grupos (colunas) e períodos (linhas).
