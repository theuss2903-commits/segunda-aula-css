# segunda-aula-css-2026-2

Aula 8 do curso — **CSS : transições e animações**.
Continuação da [primeira aula de CSS](https://github.com/claytonsouza79/primeira-aula-css-2026-2).

## O que você vai praticar

1. Personalizar cores, tamanhos e fontes no `style.css`.
2. Adicionar uma **transição** no menu (sublinhado animado).
3. Criar o botão **"Voltar ao topo"** com `transition` + `transform`.
4. Entender uma **animação com `@keyframes`** (o logo do topo já entra animado).
5. **Editar os arquivos HTML** para praticar estrutura: adicionar itens no menu, novas seções e novas `<div>`s de conteúdo.

## Atividade Prática

1. **Clone este repositório**
 Seguir orientação do Professor para clonar o seguinte repositório: 
 https://github.com/claytonsouza79/segunda-aula-css-2026-2


2. **Abra no Visual Studio Code** e comece pelo `style.css`. Procure os comentários `🎨 PERSONALIZE AQUI` — mude cores, fontes e tamanhos à vontade.

3. **Ajuste o menu** (sublinhado animado) e **crie o botão "Voltar ao topo"** conforme a explicação do professor (passo a passo nos slides `Aula-8-Transicoes-Animacoes.pptx`).

4. **Mexa nos HTMLs** — agora pode! Sugestões:
   - Adicione mais itens no `<nav class="menu">` (ex.: link "Sobre", "Contato").
   - Crie novas `<section id="...">` com um novo tema.
   - Adicione mais `<div class="card">` dentro de `<div class="cards">` com imagens da pasta `img/`.

## Arquivos

- `index.html` — página inicial com o link para as 3 turmas
- `biologia.html` — Turma A Tarde
- `historia.html` — Turma B manhã
- `quimica.html` — Turma B Tarde
- `style.css` — folha de estilos
- `img/` — imagens usadas nas páginas

## Publicar no Vercel

1. Faça um **fork** deste repositório para a sua conta do GitHub.
2. Acesse [vercel.com](https://vercel.com) → **Add New → Project**.
3. Selecione o seu fork e clique em **Deploy** (é um site estático, não precisa mudar nada).
4. Em cerca de 1 minuto o Vercel gera uma URL pública, assim que tiver pronta envie no grupo com o seu nome, caso tenha feito em dupla ou trio adicione todos os integrantes.
